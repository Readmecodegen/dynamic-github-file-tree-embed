File Icon Preview Tool

This tool helps you create and preview custom file icons before adding them to the codebase. Because your icon needs to fit perfectly inside the generated file tree, use this live sandbox to ensure your SVG sizing, coordinates, and colors look correct at every scale. Once it looks perfect, simply copy the final code below and submit your Pull Request!

Icon Input

Icons render in a 0 0 24 24 viewBox. Using x="3" width="18" leaves a 3-unit margin, which looks ideal at small sizes. Use text-anchor="middle" and x="12" to centre text.
